# ADR: Project scaffolding: Django 6.0 + Tailwind CDN

**Status**: proposed
**ID**: 38872ec3

## Description

Created Django project 'core' and app 'plantas'. Using SQLite, Python-decouple for env vars, Tailwind CDN for styling. Sage & Clay palette with Playfair Display (serif) for scientific names and JetBrains Mono (monospace) for technical data.

## Rationale

No alternatives considered — PRD mandated Django 6.0, FBVs, Tailwind CDN
