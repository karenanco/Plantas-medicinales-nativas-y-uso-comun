# ADR: Registration view as FBV with auto-login

**Status**: proposed
**ID**: 696a1417
**Proyecto**: [[projects/FloraMed-Chile/README|FloraMed-Chile]]

## Description

Implemented register() FBV in plantas/views.py using Django's UserCreationForm. On valid POST, creates user, authenticates, logs in, and redirects to home with a success message via django.contrib.messages. On GET, renders registration form with Tailwind styling matching the Sage & Clay palette.

## Rationale

FBV chosen to stay consistent with existing index() view. UserCreationForm used to avoid reinventing password validation. Auto-login improves UX by eliminating an extra login step after registration.
