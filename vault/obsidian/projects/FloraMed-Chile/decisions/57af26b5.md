# ADR: Navbar links updated with named URL references

**Status**: proposed
**ID**: 57af26b5
**Proyecto**: [[projects/FloraMed-Chile/README|FloraMed-Chile]]

## Description

Updated base.html desktop and mobile nav sections: Perfil links now point to {% url 'profile' %}, Registrarse links point to {% url 'register' %}. Updated login.html register link from # placeholder to {% url 'register' %}.

## Rationale

Hardcoded # placeholders prevented navigation. Using Django's {% url %} template tag ensures links stay valid if URL patterns change.
