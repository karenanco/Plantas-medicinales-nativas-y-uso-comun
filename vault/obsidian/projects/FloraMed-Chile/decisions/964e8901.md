# ADR: Catalog gallery as FBV with responsive Tailwind grid

**Status**: proposed
**ID**: 964e8901
**Proyecto**: [[projects/FloraMed-Chile/README|FloraMed-Chile]]

## Description

Implemented the catalog view as a function-based view (FBV) querying all Planta objects ordered by nombre_comun. Uses a responsive Tailwind grid (1/2/3/4 columns) with Sage & Clay palette. Each card shows foto placeholder, nombre_comun, nombre_cientifico in Playfair Display italic, origen badge, zona_macro badge, and uso_principal badge. Cards have hover effects (translate-y, shadow, scale image). Added a seed_data management command for 5 sample Chilean plants. Created a minimal detalle_planta stub view/template so catalog links resolve correctly.

## Rationale

CBV (ListView) — rejected to maintain FBV consistency with existing views. No grid framework — rejected in favor of Tailwind's native grid for zero-dependency responsiveness.
