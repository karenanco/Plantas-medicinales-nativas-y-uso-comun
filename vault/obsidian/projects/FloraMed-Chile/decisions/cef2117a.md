# ADR: Filters & Search implemented for catalog view

**Status**: proposed
**ID**: cef2117a
**Proyecto**: [[projects/FloraMed-Chile/README|FloraMed-Chile]]

## Description

Updated catalogo FBV to support GET-param filtering (q, origen, zona, efecto) using Django Q objects for multi-field text search and exact/icontains filters. Added Tailwind-styled filter panel above the gallery grid with search bar, three dropdowns, apply/clear buttons, active filter badge, and contextual empty state. Form uses GET method for shareable URLs.

## Rationale

Could have used django-filter library (heavier dependency), or model-level scopes (less flexible for ad-hoc filters)
